# Моделирование теплопроводности

## Идея проекта

Разработка программы для моделирования распространения тепла по объекту. Программа реализует визуализацию температурного профиля в виде анимации и тепловой карты, расчет времени для достижения заданной температуры в заданной точке. Пользователь может задавать параметры материала, граничные условия и время моделирования.

## Итерация 1

Реализация визуализации температурного профиля в виде графика и тепловой карты для одномерного объекта. Реализация расчета времени для достижения заданной температуры в заданной точке для одномерного объекта. 

## Итерация 2

Добавление анимации для одномерного объекта. Реализация функций для двумерного объекта.
